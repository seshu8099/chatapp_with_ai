# ChatMate 💬

ChatMate is a full-stack real-time messaging application. The project is structured into two main parts: a Node.js REST API backend and a client-side frontend.

## 🚀 Features

- **Real-time Messaging:** Fast and responsive chat experiences.
- **Decoupled Architecture:** Clean separation between the client and the server.
- **Environment Configuration:** Easily manage secrets and configurations using `.env` files.

## 📁 Project Structure

- `/Backend` - Contains the Node.js server, API routes, controllers, and configuration files.
- `/Frontend` - Contains the client-side user interface and assets.

## 🛠️ Tech Stack

**Backend:**
- Node.js & Express (API & Server)
- dotenv (Environment Variable Management)
- nodemon (Development server monitoring)
- cors (Cross-Origin Resource Sharing)
